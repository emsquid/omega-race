## Installation

Get the project and install dependencies
```bash
git clone https://github.com/emsquid/omega-race

cd omega-race

pip3 install -r requirements.txt
```

And enjoy yourself 
```bash
python3 main.py
```

You may need the `.env` to run the game with all options, ask me :)
